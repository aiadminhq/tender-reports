# 📋 每日政府採購標案配發系統

自動蒐集政府採購網（web.pcc.gov.tw）及北醫附屬機構聯合採購（cmd.tmu.edu.tw）標案，
經 AI 媒合評分後每日自動發布至此，供業務團隊快速掌握投標機會。

## 🔗 索引頁（每日更新）
> https://aiadminhq.github.io/tender-reports/

## 篩選條件
- 📍 **地區**：北部（台北、新北）
- 🏗️ **類別**：工程類為主
- ❌ **排除**：最有利標
- 💰 **預算**：8,000 萬以下
- ⏰ **截止**：排除 7 天內截止（備標時間不足；⭐最優先案豁免）

## 關鍵字
裝修工程 / 改善工程 / 整修工程 / 裝潢 / 汰換

## 每日執行時間
🕗 每天 08:00（台灣時間）由 GitHub Actions 自動觸發
